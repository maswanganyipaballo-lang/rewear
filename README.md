# ReWear

A second-hand fashion marketplace built with PHP, MySQL, HTML, CSS, and Bootstrap 5.

## Features
- User registration and login (buyer, seller, admin roles)
- Product listings with image uploads
- Shopping cart and checkout via PayFast
- Seller dashboard to manage listings
- Admin dashboard to manage users, products, and orders

## Technologies Used
- PHP
- MySQL
- HTML / CSS / JavaScript
- Bootstrap 5
- XAMPP (local development)

## How to Run Locally
1. Clone this repository into your XAMPP htdocs folder
2. Import `setup.sql` into phpMyAdmin as `rewear_db`
3. Update `db.php` with your database credentials
4. Visit `http://localhost/rewear`
